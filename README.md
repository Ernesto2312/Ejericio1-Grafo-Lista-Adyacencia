# Ejercicio 1 – Grafo con Lista de Adyacencia

## 📌 Descripción
Implementación de un **grafo usando listas de adyacencia**, permitiendo representar relaciones entre nodos de forma eficiente.  
Este ejercicio forma parte de la asignatura de Estructuras de Datos.

## 🧩 Funcionalidades
- Crear nodos del grafo
- Agregar aristas (dirigidas o no dirigidas)
- Mostrar la lista de adyacencia
- Buscar conexiones entre nodos
- Eliminar nodos o aristas (si aplica)

## 🛠️ Tecnologías
- Lenguaje: **Java**
- Paradigma: Programación orientada a objetos

## 📂 Estructura del proyecto
- Grafo.java
- Nodo.java
- Main.java

 
## ▶️ Ejecución
Compilar
javac src/*.java

## Ejecutar:
java src/Main

## 📝 Ejemplo de salida:
Nodo A -> B, C
Nodo B -> C
Nodo C -> A
